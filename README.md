# Projeto IHC

> Protótipo de interface para disciplina optativa de Interação Humano-Computador.

## Visão geral

Este repositório contém um protótipo web estático desenvolvido para a matéria optativa de IHC. O foco do projeto é demonstrar uma experiência de login limpa, usabilidade básica e organização de um projeto front-end simples.

## Objetivos

- Apresentar um protótipo visual funcional com HTML e CSS.
- Garantir uma interface clara e responsiva para dispositivos de diferentes larguras.
- Priorizar organização, simplicidade e legibilidade do código.
- Demonstrar conhecimentos básicos de design de interface e estruturação de projeto.

## Principais características

- Layout minimalista e fácil de entender.
- Tela de login com campos de entrada e botões de ação.
- Estrutura de arquivos leve e bem organizada.
- Uso de estilos CSS puros, sem dependência de bibliotecas externas.

## Estrutura do projeto

- `index.html` — arquivo principal que exibe a interface.
- `style.css` — estilos e regras visuais do protótipo.
- `assets/` — imagens e recursos usados no projeto.
- `.gitignore` — exclusão de arquivos sensíveis e específicos do ambiente.

## Como executar

1. Abra o diretório do projeto.
2. Execute `index.html` em um navegador moderno.
3. Navegue pelas telas e analise o layout apresentado.

## Observações

Este trabalho é uma implementação simples, pensada para uma apresentação acadêmica. A proposta é focar em design de interface e estrutura de projeto, sem complexidade funcional ou backend.
